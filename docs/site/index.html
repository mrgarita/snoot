Snood 風ブラウザゲーム「Snoot」の開発記録（各 step の文書一覧）

このサイトについて

かつて Mac で人気を博したゲーム Snood をオリジナル素材で再現する ブラウザゲーム「Snoot」の開発過程を、step ごとに文書として残したもの。 開発フローは次の 5 step（project.txt で定義）。

  1. Snood がどんなゲームかを調査する
  2. 調査・フィードバックに基づき Snoot を制作する
  3. 人間が実際にプレイして面白さと再現度をチェックする
  4. フィードバックに基づき step2→3 を反復する（バージョンを上げながら）
  5. 一般公開用サイトを作成し XServer に本番デプロイして完了
